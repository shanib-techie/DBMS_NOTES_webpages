# DBMS Notes & SQL Practice Resources

A complete collection of **DBMS notes, SQL practice queries, and concept explanations** designed for students preparing for exams, assignments, and interviews.

## 📌 What's Included

- DBMS Notes in HTML format
- SQL Joins practice with examples
- ER Model concepts
- Three Schema Architecture explanation
- Student Society DBMS project queries
- Practice SQL files for hands-on learning

## 📂 Files Overview

| File Name | Description |
|----------|-------------|
| `dbms_notes.html` | Main DBMS notes |
| `DBMS_Notes_2024.html` | Updated DBMS notes version |
| `sql_joins_complete.html` | SQL joins explanation with examples |
| `join_practice.sql` | SQL joins practice queries |
| `er_model_concepts.html` | ER model concepts |
| `three_schema_architecture.html` | 3-schema architecture explanation |
| `all_queries_of_stu_soc.sql` | Student society SQL queries |

## 🚀 How to Use

1. Open HTML files in browser  
2. Run `.sql` files in MySQL or any SQL environment  
3. Practice queries and revise concepts  

## 🎯 Purpose

This repository was created to make DBMS easier for students by combining:
- theory notes
- practical SQL queries
- exam-focused concepts
